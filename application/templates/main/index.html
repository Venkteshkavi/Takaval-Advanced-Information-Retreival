How does takaval work?

Crawl News Articles

Data Collection

This module helps in collecting news articles from the major news websites. Initially a crawler crawls all the relevant urls for a given query. This url serves as an index to scrape article contents. The crawler that scrapes the article's content is tailored to the article's source website.

Extract Events From Articles

Data Extraction

The module handles tasks like topic classification, extracting events, locations, actors involved, summarizing the article and attribute whether the event has relation towards elections.

Store Data For Processing

Data Storage

A MySQL database is used to store all the crawled articles. This is then used by the Data Extraction module to extract events and do other filtering operations on the article.

Web App To View Results

Web Application

The takaval web application helps to view the crawled data, view analytics, heatmap of events with its location, view the detailed summary of events and view if the events are related to election.
